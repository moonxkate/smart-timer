# Таймер

Минималистичное веб-приложение для управления временем с двумя режимами: секундомером и обратным отсчётом.

## Возможности

- **Секундомер** — запуск, пауза, сброс
- **Обратный отсчёт** — настраиваемый таймер от 1 до 90 минут с паузой и перезапуском
- **Уведомления** — браузерные push-уведомления по окончании обратного отсчёта со звуковым сигналом
- **Точный отсчёт времени** — оба режима используют системное время вместо интервалов, работают корректно в фоновых вкладках

## Стек

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- Web Notifications API
